Основная цена

Начальная цена - 7000р

В стоимость включено:

  • Создание отдельной категории для организации на сервере.
  • Создание 5 различных профессий с суммарным лимитом в 15.
  • Выдача стандартных оружий фракции, если таковые имеются.
  • Стандартный набор характеристик для каждой профессии.
  • Выбор одного оружия из имеющихся на сервере для каждой профессии.
  • Звание при покупке - для командира организации LT. Остальные звания командир организации вправе выдать сам.

Внимание!

Все добавленные профессии включают в себя то же самое, что и основные профессии организации.

При добавлении профессии, суммарный лимит не увеличивается.

Все остальные характеристики, оружие, модели и т.п. рассчитываются исходя из ценников для личных профессий, для каждой профессии отряда по отдельности, но со скидкой в 30%.
